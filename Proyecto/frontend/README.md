# Frontend - Sistema de Gestión de Medios

Frontend administrativo en React para gestionar Géneros, Directores, Productoras, Tipos y Media.

## Funcionalidad
- Navegación por módulos con React Router.
- Formularios para crear y editar registros.
- Listados para consultar, editar y eliminar.
- Publicación de media enlazada a catálogos activos.
- Dashboard inicial con resumen de registros.

## Tecnologías
- React 19.
- React Router.
- Axios.
- Vite.

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Vista previa
```bash
npm run preview
```
